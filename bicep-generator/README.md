# bicep-generator

A **Python CLI** that can generate Azure Bicep files for **any** Azure resource type.

## Features
- Reads a high‑level JSON/YAML description of resources.
- Pulls the latest Azure Resource Provider schemas from the public repo.
- Uses Jinja2 templates (≈800 snippets) to render each resource.
- Automatically resolves dependencies (e.g., VNet → Subnet).
- Generates a matching `.bicepparam` file for parameters.
- Optional diagram generation via the existing `bicep-diagrams.py`.

## Quick start
```bash
# clone (already in your workspace)
cd bicep-generator
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# generate templates (first run only)
python scripts/generate_templates.py

# example generation
python -m src.cli -i examples/simple_vnet.json -o out/vnet.bicep
```
